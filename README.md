# LightAlarm - световой будильник

## О проекте
Проект создания светового будильника на базе чипа MSP430G2553

                MSP4302553
             -----------------
         /|\|              XIN|-
          | |                 |
          --|RST          XOUT|-
            |                 |
            |             P1.6|-->PWM LED
            |                 |
            |             P2.1|-->PWM LED
            |                 |
            |             P2.4|-->PWM LED
            |                 |
            |     P1.2/UCA0TXD|------------>
            |                 | 115200 - 8N1
            |     P1.1/UCA0RXD|<------------
